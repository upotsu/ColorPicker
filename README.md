# RGB分析アプリ（Color Picker）
URL：https://github.com/upotsu/ColorPicker/releases

# ■ 目的

キャラクターデザイン作成時に、参照素材の配色（RGB / HSV / HEX）を正確に取得するためのカラーピッカーを作成。

Web上のカラーピッカーは存在するが、画像をインターネットにアップロードできない環境や機密素材を扱う制作現場などのオフライン環境でも使用できるよう、ローカル完結型ツールとして開発。

# ■ 概要

本アプリは、ローカルPC上で画像を読み込み、
マウス操作によりピクセル単位で色を取得できるデスクトップカラーピッカーです。

取得可能な色情報：

RGB値

HSV値

HEXコード（#RRGGBB形式）

# ■ 主な仕様

ローカル画像ファイル（PNG / JPG 等）を読み込み可能

マウス位置のピクセル色をリアルタイム取得

RGB / HSV / HEX を同時表示

外部通信なし（完全オフライン動作）

データ保存なし（情報漏洩リスク低）

# ■ 使用環境

OS：Windows 10 / 11（64bit）

Python：不要（exe単体で動作）

ネット接続：不要

※ 本アプリは PySide6 / NumPy を使用してビルドされています。

# ■ セキュリティ設計

画像データはローカルメモリ上のみで処理

外部サーバー通信なし

アップロード機能なし

ファイル保存機能なし

そのため、制作中の非公開素材の配色確認用途に適しています。
